
Object - as the name suggests it is an object like car,books,dogs,cats.They are real world entities that have their own attributes and methods.
Class - a blueprint of an object,or representation of an object. It holds the attributes and methods of the object.
                      Attributes - what the object is like (colour,size)
                      Methods - its functionalities (eat,sleep)
